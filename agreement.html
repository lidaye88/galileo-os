一、术语定义

  • 本平台：指思为交互科技运营的 Galileo OS 工业智能开放平台，包括网站、移动端及相关服务。
  • 运营方：指四川思为交互科技有限公司，注册地成都市高新区盛安街 401 号凯旋南城 B 座 1502。
  • 用户：指通过注册或以其他方式使用本平台服务的自然人、法人或其他组织。
  • 内容：指用户在使用服务过程中上传、发布、存储的数据、模型、Agent、Skill 及其他数字资产。
  • OPC：指与本平台合作的独立实施专家（运营顾问），负责行业方案落地交付。

二、服务内容

本平台为制造业提供工业智能操作系统及相关服务，具体包括但不限于：

  • 工业 Agent 市场：提供生产调度、预测维护、质量分析等数字化员工；
  • Skill 市场：提供可组合的工业能力单元（排产算法、缺陷识别等）；
  • 应用市场：提供 MES、EAM、QMS 等工厂业务系统；
  • 知识库市场：提供工艺、故障、安全等行业知识包；
  • 开发者工具：支持用户自定义开发并上架 Skill、Agent、应用；
  • OPC 生态：连接独立实施专家与工厂需求。

运营方有权根据业务发展调整服务范围，调整前将在平台公告。

三、账号注册与管理

3.1 注册资格

用户须为具备完全民事行为能力的自然人，或依法成立的法人/组织。未成年人不得注册。

3.2 注册信息

  • 用户应提供真实、准确、完整的注册信息（姓名、手机号、企业名称等）；
  • 信息发生变化时应及时更新；
  • 因信息不实导致的后果由用户自行承担。

3.3 账号安全

  • 用户须妥善保管账号、密码及 API 密钥，因泄露造成的损失由用户承担；
  • 发现账号被盗用应立即联系客服（400-058-0097）；
  • 每个手机号仅可注册一个账号，账号不可转让、出借。

四、用户行为规范

用户在使用本平台服务时，应遵守中华人民共和国法律法规，不得利用本平台从事下列行为：

  • 上传或传播违法、有害、侵犯他人权益的内容；
  • 未经授权访问、篡改、删除他人数据；
  • 对本平台进行反向工程、反编译、破解或干扰正常运行；
  • 利用本平台开发恶意软件或进行网络攻击；
  • 批量抓取、爬取平台数据用于商业转售（未经授权）；
  • 其他可能损害运营方、其他用户或第三方合法权益的行为。

5.1 计费模式

本平台部分服务收费，具体计费方式包括：按调用次数计费、按设备/工位计费、订阅制、节能分润等，以各服务页面公示为准。

5.2 费用支付

  • 用户应在使用付费服务前完成充值或订阅；
  • 账单一经生成即为应付，已支付费用除法律另有规定外不予退还；
  • 价格调整将提前 7 天公告，已订阅用户在当前周期内不受影响。

5.3 发票

用户可凭付款记录向运营方申请开具增值税发票，联系邮箱 waynezheng@idmakers.cn。

6.2 用户内容

  • 用户授予运营方在提供服务范围内使用、存储、传输内容的非独家许可；
  • 用户上架至市场的内容，按《上架协议》约定分润。

用户使用平台工具开发的 Skill、Agent、应用，其知识产权归开发者所有，但不得侵犯运营方或第三方的在先权利。

九、违约与处理

用户违反本协议的，运营方有权视情节采取下列措施：

  • 警告、限制功能、暂停服务；
  • 删除违规内容、冻结账号及资产；
  • 追究法律责任并要求赔偿损失。

十、协议变更与终止

10.1 协议变更

运营方有权根据法律法规或业务需要修订本协议，修订后将在平台公告。用户继续使用服务即视为接受修订；不接受则可终止使用。

10.2 服务终止
